### Projeto de Gerenciamento Bancário com Spring Boot
### Este é um projeto desenvolvido em Spring Boot para demonstrar um sistema de gerenciamento bancário, com planos de escalar.

### 🚀 Tecnologias Utilizadas
### Java 21
### Spring Boot 3.5.4
### PostgreSQL (9.5)

### Este projeto ainda está em desenvolvimento, será implementado novas funcionalidades

### 🔐 Implementação de autenticação de dois fatores
### 📊 Criação de relatórios personalizados
### 🆔 Adicionar validação de CPF para identificar se é válido ou falso
### ⚠️ Implementar tratamento com try/catch para evitar que o programa quebre com entradas inválidas
